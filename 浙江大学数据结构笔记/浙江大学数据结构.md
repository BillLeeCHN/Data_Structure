## 1.1 打印数字

```c
见程序

```



```c
// for循环方法
// 推荐
void Print1(int N)
{
    int i;
    for (i = 1; i <= N; i++)
    {
        printf("%d\n", i);
    }   
}
// 递归方法
// 缺点：占用大量内存，当N太大是会爆掉，且耗时间
// 不推荐
void Print2(int N)
{
    if (N)
    {
        Print2(N - 1);
        printf("%d\n", N);
    }
}
```
